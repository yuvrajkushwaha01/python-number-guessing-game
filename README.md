# 🎯 Number Guessing Game (Python)

A simple console-based Number Guessing Game built using Python.

## 📌 Project Description

This project is a mini Python game where:
- The system generates a random number between 1 and 100
- The user selects a difficulty level
- The user tries to guess the number within limited attempts

## 🚀 Features

✔ Difficulty Levels (Easy, Medium, Hard)  
✔ Limited Attempts System  
✔ Win / Lose Condition  
✔ Random Number Generation  
✔ Clean and Beginner-Friendly Code  

## 🛠 Technologies Used

- Python
- Random Module
- Loops (while)
- Conditional Statements (if-else)

## ▶ How to Run

1. Clone this repository:
   ```
   git clone https://github.com/yuvrajkushwaha01/number-guess-game-python.git
   ```

2. Navigate to the folder:
   ```
   cd number-guess-game-python
   ```

3. Run the program:
   ```
   python main.py
   ```
